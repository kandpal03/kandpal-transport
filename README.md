# 🚚 Kandpal Transport

A full-stack logistics and transport booking project built with React, Node.js, Express, Docker, Jenkins CI/CD, Docker Hub, Kubernetes and ArgoCD.

---

## 📌 Project Overview

Kandpal Transport is a logistics and transport booking web application where users can explore truck options, calculate transport pricing, submit customer inquiries, and track demo shipments.

This project is built as an interview-ready full-stack DevOps project with frontend, backend, containerization, CI/CD pipeline and Kubernetes deployment.

---

## ✨ Features

- Premium transport landing page
- Truck options with images
- Truck selection flow
- Price calculator
- Live tracking demo
- Customer inquiry form
- Admin portal demo
- Customer data view
- Node.js backend APIs
- Dockerized application
- Jenkins CI/CD pipeline
- Docker Hub image push
- Kubernetes deployment
- ArgoCD GitOps deployment
