# Kandpal Transport

A full-stack logistics and transport booking project built with React, Node.js, Express, Docker and Kubernetes.

## Features

- Premium transport landing page
- Truck options with images
- Price calculator
- Live tracking demo
- Customer inquiry form
- Admin portal demo
- Node.js backend APIs
- Docker support
- Kubernetes deployment files

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js

### DevOps
- Docker
- Kubernetes
- Jenkins pipeline planned
- Docker Hub image push planned
